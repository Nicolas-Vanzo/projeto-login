<h1 align="center"> Projeto Login</h1>

### Descrição

Projeto criado na aula da disciplina de programação Front-End, na Unicesumar
Campus de Londrina. O professor [Leonarndo Rocha](https://github.com/leonardorocha) introduziu os conceitos de framework,
apresentando
o Bootstrap e alguns exemplos disponiveis nesse framework, e git, realizando o passo-a-passo de configurações do ambiente de
desenvolvimento

![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

### Ferramentas Utilizadas

- HTML
- CSS3
- Bootstrap
- Git

### Como foi desenvolvido

Os codigos foram editados pelo vscode e conforme as ediçoes fui usando:
- git add .
- git commit -m 
- git push

para subir os arquivos no github
